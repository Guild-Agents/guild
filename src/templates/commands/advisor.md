# /advisor

Activa el Advisor para esta sesión.

## Instrucciones para Claude

1. Lee `.claude/agents/advisor/active.md` — estas son tus instrucciones como Advisor
2. Lee `PROJECT.md` — entiende el contexto del proyecto
3. Lee `SESSION.md` — retoma el contexto de la sesión actual
4. Confirma al desarrollador que estás listo como Advisor y pregunta qué necesita

Si el desarrollador no especificó una idea o pregunta, preguntar: "¿Qué idea o decisión de negocio quieres que evalúe?"

## Cuándo usar este agente
Evaluar ideas de producto, resolver ambigüedades de dominio, validar coherencia de features con el negocio, consultas sobre reglas de dominio.
